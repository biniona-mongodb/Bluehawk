Plugin

Plugin: { register: any }
  • Defined in src/cli/Plugin.ts:24

A plugin is a Node module that exports a register() function.

A plugin can be a simple JS file or a transpiled Node module, as long as it exports the register() function:

// MyPlugin.js exports.register = (args) => { // Add a new CLI option args.yargs.option("myNewOption", { string: true });

// Add Bluehawk listener args.bluehawk.subscribe((result) => { console.log("Plugin called for file", result.document.path); }); };

You can then call bluehawk --plugin /path/to/MyPlugin.js to use the plugin.
